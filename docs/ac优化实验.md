> Baseline: Plain MLP 2L/256h, β=1.0, 50K steps → ref_mse=0.282

### Phase 1 关键结论

- **Residual connections 是最大的单一改进** (42x)，远超其他因子
- LayerNorm (20x) > hidden_512 (12x) > SiLU (11x) > beta_0.1 (10x)
- 学习率 1e-4 单独就有 28x 提升（但在 residual 上不叠加）
- Beta 太低 (0.01) 完全发散；太高 (5.0) 限制 actor 学习

### Phase 2 关键结论

- **res + β=0.3 是最优组合** (ref_mse=0.009)
- Residual 网络需要更高的 β：β=0.05 在 plain MLP 上好，在 residual 上发散

- 大 batch (1024) 导致发散；小 batch (128) 有帮助
- 非 residual 的最优：silu + w512 + β=0.1 (ref_mse=0.033)

## 最终最优配置

```yaml
architecture:
  type: ResidualMLP
  hidden_dim: 256
  num_layers: 3
  activation: relu
  layer_norm: false
  residual: true

training:
  beta: 0.3          # BC regularization weight
  lr: 3e-4           # both actor and critic
  tau: 0.005         # target network soft update
  gamma: 0.99        # discount factor
  batch_size: 256
  actor_update_interval: 2
  gradient_steps: 50000  # sweet spot, do NOT exceed 100K
  ref_dropout_p: 0.5
  fixed_std: 0.05

result:
  ref_mse: 0.008-0.011  # (vs v1 baseline 0.334, 30-42x improvement)
  q_gap: -0.011          # healthy negative
  actor_params: ~800K
  critic_params: ~1.5M   # twin Q
```

## 核心 Insights

1. **Residual connections 是最大的单一改进** (42x)，远超 SiLU (11x)、wide (12x)、LayerNorm (20x)
2. **改进因子不叠加**: Residual + SiLU + wide + LN 的"全叠加"不如单独 Residual
3. **Residual 网络需要更高 β**: plain MLP 最优 β=0.05-0.1，Residual 最优 β=0.3-0.5
4. **训练时间有 sweet spot**: 50K-100K 步最优，200K+ 在固定 buffer 上过拟合
5. **大 batch (1024) 有害**: 导致训练发散，128-256 最稳定
6. **学习率**: 默认 3e-4 即可，1e-3 有害，1e-4 在长训练时稍好但短训练无优势
7. **Ref dropout 0.5 最平衡**: 0.0 短期好但可能泛化差，0.7 伤害性能
8. **Base 模型 > SFT 模型做 RL token**: base 表征更可压缩 (loss 0.015 vs 0.026)，AC 效果也更好 (ref_mse 0.011 vs 0.043)
